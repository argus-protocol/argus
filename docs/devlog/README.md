# Argus dev log

Public, written log of what Argus is shipping, what is being deferred, and
what is being learned. Entries land alongside milestones — weekly during
active grant periods, less regularly between.

Format: short, factual, opinionated where it matters. No marketing.
Numbers where there are numbers, trade-offs where there are trade-offs.

## Why public

The EF ESP proposal commits to open design documents, architecture decision
records, and post-mortems published in this repository. This is where those
live. Weekly cadence is aspirational; material over schedule.
